EXPRESSIONS

Unless specified, all operators and the result are the same size 
All instructions are unsigned by default
Uppercase arguments must be known at compile time

OPR ARGUMENTS DESCRIPTION
=== ========= ===========
'   SZ p      Dereferences p as size SZ
.   val SZ    Truncates/zero-extends val to SZ bytes.
`.  val SZ    Truncates/sign-extends val to SZ bytes
+   x y       Adds x and y
-   x y       Subtracts y from x
*   x y       Multiplies x and y
/   x y       Divides x by y
`/  x y       Signed divides x by y
%   x y       Division remainder of x / y
`/  x y       Signed divison remainder of x / y
&   x y       ANDs x and y
|   x y       ORs x and y
^   x y       XORs x and y
~   x         NOTs x
<<  x b       Shifts x b bits left.
>>  x b       Shifts x b bits to the right
`>> x b       Shifts x b bits to the right and sign-extends
!   x         1 if x is 0, 0 otherwise
==  x y       1 if x equals y, 0 otherwise
<   x y       1 if x is less than y, 0 otherwise
>   x y       1 if x is greater than y, 0 otherwise

STATEMENTS

... repeats that argument. 
( )... repeats all arguments inside the parinthesis.
Uppercase arguments must be known at compile time

OPR ARGUMENTS DESCRIPTION
=== ========= ===========
[   SZ ID...  Creates new variable(s) of size SZ pointed to by ID(s)
]   ID...     Deletes all specified variables
(   ,val...   Puts all values into function storage
)   ,(S p)... Gets values from function storage and puts them in p(s) of size S(s)
{   DST       Saves return address and jumps to DST. Nested calls are supported.
}             Jumps to the last saved return address
@   LABEL     Defines a global/external label
:   LABEL     Defines a local label
;   LABEL     Deletes a local label
?   cnd dst   Jumps to dst if cnd is non-zero
=   dst val   Puts val in dst
$   LBL V...  Declares data segment of Vs, stored in order. Supports multi-char strings
